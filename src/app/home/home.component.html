# Зачем это все нужно?

Мы создали базу проекта на Angular для того, чтобы у нас всегда была возможность быстро развернуть «болванку» проекта со всеми уже подключенными и настроенными инструментами.

# Что используется под капотом?

  • Angular - angular.io
  • Bootstrap 4 - getbootstrap.com
  • Firebase - firebase.google.com

# Как этим пользоваться?

Клонируем репозиторий затем разворачиваем модули.

npm install
ng serve  # Запускаем Angular на localhost:4200

На типовой странице можно посмотреть как заюзаны основные элементы типографики.

Готовая авторизация, регистрация и восстановление доступа с независимым сервисом хранения данных.
